Check the numbers

Write a program that reads an array of integers and two numbers n and m. The program is to check that n and m never occur next to each other (in any order) in the array.

Input data format

The first line contains the size of an array.

The second line contains elements of the array.

The third line contains two integer numbers n and m.

All the numbers in the same line are separated by the space character.

Output data format

The result is a single value, that is, "YES" if n and m never occur to each other, otherwise, "NO".


Sample Input:
3
1 2 3
2 3

Sample Output:
NO


Sample Input:
3
1 2 3
3 4

Sample Output:
YES



Memory limit: 256 MB
Time limit: 8 seconds
