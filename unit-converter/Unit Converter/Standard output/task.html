Description

By the end of this project, you will have a program that can convert length, weight, and temperature from one common unit of measurement to another, such as inches to meters, kilograms to pounds, and so on.

At this stage, you will write a program that converts some predefined values and predefined units.

Output example

The output of your program should exactly match the example below.

145 centimeters is 1.45 meters
2 miles is 3.2187 kilometers
5.5 inches is 139.7 millimeters
12 degrees Celsius is 53.6 degrees Fahrenheit
3 pounds is 1.360776 kilograms
