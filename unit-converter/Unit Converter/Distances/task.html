Description

At this stage, you will write a program that can convert common units of length to meters.

The conversion should follow these rules:

  1. One meter equals 1 meter.
  2. One kilometer equals 1000 meters.
  3. One centimeter equals 0.01 meters.
  4. One millimeter equals 0.001 meters.
  5. One mile equals 1609.35 meters.
  6. One yard equals 0.9144 meters.
  7. One foot equals 0.3048 meters.
  8. One inch equals 0.0254 meters.

The user should input a number and a unit of measurement. These following units are allowed:

  1. For meters, the user can input "m", "meter", or "meters".
  2. For kilometers, the user can input "km", "kilometer", or "kilometers".
  3. For centimeters, the user can input "cm", "centimeter", or "centimeters".
  4. For millimeters, the user can input "mm", "millimeter", or "millimeters".
  5. For miles, the user can input "mi", "mile", or "miles".
  6. For yards, the user can input "yd", "yard", or "yards".
  7. For feet, the user can input "ft", "foot", or "feet".
  8. For inches, the user can input "in", "inch", or "inches".

Also, the units of measurement should be case insensitive. That means that "Meters", "METERS", and "mEtErS" should work correctly as well as "meters".

The output should contain the full names of units of measurement. Additionally, if the value is 1.0, the name of the unit must be singular. For example, "1.0 meter", but "1.1 meters".
