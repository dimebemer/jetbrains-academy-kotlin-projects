Description

At this stage, you will write a program that can convert weight from one common unit to another. The user should also be able to convert values from any of the following units of weight to any other.

Hint: Hard-coding all the possible conversion ratios is possible, but not very efficient. A smarter way would be to convert the input value to one intermediate ‘hub’ unit (for example, meters) using the general conversion table below, and then convert that intermediate value to the end target value.

The conversion should follow these rules:

  1. One gram equals 1 gram.
  2. One kilogram equals 1000 grams.
  3. One milligram equals 0.001 grams.
  4. One pound equals 453.592 grams.
  5. One ounce equals 28.3495 grams.

The user should input a number, then the source unit of weight, then some random word (for example, "in" or "to", but it can be any other word), and finally the target unit of mass. In addition to the units above, the following units should also be allowed:

  1. For grams, the user can input "g", "gram", or "grams".
  2. For kilograms, the user can input "kg", "kilogram", or "kilograms".
  3. For milligrams, the user can input "mg", "milligram", or "milligrams".
  4. For pounds, the user can input "lb", "pound", or "pounds".
  5. For ounces, the user can input "oz", "ounce", or "ounces".

The program should keep processing user input until they enter “exit”.
