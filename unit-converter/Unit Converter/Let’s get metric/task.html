Description

At this stage, you will create a program that converts kilometers to meters. The user inputs a number of kilometers, and the program outputs its equivalent in meters. (Only a single input is required.)

Note that one kilometer equals exactly 1000 meters.

The program should ask the user about kilometers only once, not infinitely.

Output examples

The examples are shown below:

Enter a number of kilometers: 25
25 kilometers is 25000 meters
Enter a number of kilometers: 12
12 kilometers is 12000 meters
