Board game basics: Game

You want to create a game and you decided to keep the playing field in some structure. You want to have a single field for the game, so you decided to store it in a PlayingField object. In addition, you added a Size object inside that class to store the field measurements. This is a 2D game, so width and height properties will be enough to define the field size. But your field can change, so you need to be able to resize it using the changeSize() function with the following signature: 

fun changeSize(width: Int, height: Int)

The playing field size cannot be negative. If that happens, set the value to 0.

Note that the changeSize() function must be a member of the Size object.
